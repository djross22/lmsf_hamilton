ASW Icon Set ReadMe
--------------------------------------------------------------------------------------------------------------------

Lizenzen:
------------
  Die meisten Icons  in diesem Icon Set, oder Abwandlungen davon, entstammen aus dem Oxygen Icon Set Version 4.3.1 des KDE Projektes und sind
  unter einer Creative Commons Lizenz frei verfügbar. Bei jedem Projekt, welches Icons aus dem ASW Standard Icon-Set verwendet, muss die Datei 
  Legal.txt oder deren Inhalt mitgeliefert werden, resp. über ein Menü oder entsprechend dargestellt werden können.

  **********************************************************************************************************************************************
  ***WICHTIG***: Solange von der IP-Abteilung der Hamilton noch kein offizielles Statement zur Verwendung der Oxygen Icons vorliegt, dürfen    *
  *              die Oxygen-Icons nur dynamisch in eine Applikation gelinkt werden, so dass sie jederzeit durch Ersatz-Icons über              *
  *              das Dateisystem ausgetauscht werden können, ohne die Applikation neu kompilieren zu müssen.                                   *
  **********************************************************************************************************************************************

Verwendung:
------------
  Alle Icons sind in der Grösse 32x32 pixel mit 96 DPI im Dateiformat PNG vorhanden. Bei allen anderen Grössen sind nicht immer alle Icons vorhanden.

  Falls ein Icon in der gewünschten Grösse noch nicht vorhanden ist, oder noch ein Icon fehlt, bitte den Mainainer 
  der letzten Revision per Mail kontaktieren. Icon Vorschläge für das ASW Standard Icon Set können folgendem Icon-Set entnommen werden: 
  G:\Robotics\Software\Application Software\Application Details\HAM IconSet\oxygen-icons-4.3.1

  Im Mail bitte folgende Angaben machen:
  - Quell-Pfad/Name der Originaldatei in oxygen-icons-4.3.1
  - Ziel-Pfad/Name der Datei im ASWIconSet
  - gewünschte Grössen im Format HÖHExBREITE. Es werden nur quadratische Grössen unterstützt.

  Viele Icons im Ordner "liquidhandling" stammen von Hamilton. Diese benötigen keine spezielle beschreibung, da der Verwendungszweck 
  jedes Icons dem Verwendungszweck in der Venus-Software gleichkommt. Falls ein Icon aus der Venus Software gebraucht wird, welches
  in diesem Ordner nicht aufgeführt ist, erfolgt eine Anfrage analog zum Vorgehen bei Icons aus dem Oxygen Icon Set. Statt einen
  Quell-Pfad/Namen anzugeben, kann der Name der Funktion in der Venus Software angegeben werden, welcher dieses Icon verwendet. Alternativ
  kann auch ein Screenshot des Icons per Mail geschickt werden.

  Das Hamilton-Logo darf explizit nicht verändert werden. Es liegt deshalb in allen benötigten Grössen als PNG-Datei vor.
  Das Hamilton-Logo als ICO-Datei liegt im Ordner "ICO" als Ebenen-basierte ICO-Datei vor. D.h. Windows sucht sich selbst 
  die korrekte Darstellungsgrösse aus den vorhandenen Logos aus. Zudem wurde das Hamilton-Icon speziell für die Grössen 
  - 16x16 (Für Fensterdekoration von Applikationen, Detailansicht im Explorer, Eintrag im Startmenü usw.)
  - 32x32 (Für Desktopansicht, Symbolansicht im Explorer)
  - 48x48 (Für Kachelansicht im Explorer)
  optimiert und mit einem weichen Rand versehen, damit sich das Icon in jeder Grösse schöner in die Arbeitsfläche integriert.

  Die Status-LED- und die Labware-Icons sind eine Eigenkreation von Romano Sommer. Diese Icons können durch den Betreuer der letzten Version 
  des ASWIconSet nach belieben verändert werden. Änderungswünsche können per Mail beantragt werden.
